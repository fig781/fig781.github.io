Country Currency Information Search

Vue.js HTML CSS Javascript

This application uses the forex API from https://fcsapi.com/. You can add countries to the search queue and then search for all of them at once. For those countries, you will be shown the details of that country’s national currency.

What I Learned:

This is the first application that I have made using Vue. This project taught me all the basics of Vue. It took a bit of time to fully understand the Vue directives (v-on, v-bind etc…) and how the script portion of the component was laid out. Once I understood these parts of Vue, then the rest was relatively easy. The concept of components and state are ubiquitous with React, but I enjoy Vue’s approach better. I found that I could create the same types of components that I have made in React, but in Vue, the same components had fewer lines of code. Overall, I am very happy with Vue. I plan on using it in all my future projects.

Learnfluent - Chrome Extension for Foreign Languge Learning

HTML CSS Javascript

Learnfluent is a chrome extension that helps you learn a foreign language. First, you set the language that you want to learn. French, German, Spanish and Russian are the four languages you can choose from. Then, each day, you are given a new commonly used word in the language of your choice for you to practice.

What I Learned:

I made this project primarily to learn how to integrate an existing API into my own work. Most of the extension changes dynamically with the day of the month and users can save their language preferences. Learning how to use the chrome API to create these functionalities was very exciting. I originally overengineered the extension. My first iteration was complicated and used more of the API functionality then was needed. Learning how to simplify the project design was also an important learning moment.
